Variables:
- Declaración: var, let, const y sus diferencias
- Reglas de nomenclatura (camelCase, sin espacios, sin caracteres especiales)
- Asignación e inicialización de variables
- Tipado dinámico en JavaScript

Funciones Nativas:
- console.log()
-  parseInt() y parseFloat() / Number()
- Concatenación con + y Template Literals ( ` con ${})
- alert(), confirm(), prompt()

Expresiones Aritméticas:
- Operadores básicos: suma +, resta -, multiplicación *, división /
- Módulo % y exponente **
- Operadores de incremento ++ y decremento --
- Operadores de asignación compuesta: +=, -=, *=, /=, %=

- Igualdad == vs igualdad estricta ===
- Desigualdad != y !==
- Mayor/menor que: >, <, >=, <=
- Operadores lógicos: AND &&, OR ||, NOT !

Sentencias Condicionales:
- if / else if / else
- Condicionales con operadores && y ||
- Operador ternario condición ? valor1 : valor2 (pendiente)
- switch con case, break y default (pendiente)

Funciones:

- Retorno de valores con return
- Scope: ámbito global vs. local
- Funciones anónimas asignadas a variables
- Arrow Functions (funciones flecha) con sintaxis simplificada (pendiente)
- Callbacks: funciones como argumentos de otras funciones
